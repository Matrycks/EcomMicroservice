# Ecommernce Order Management Application (Microservice)

A demo ASP.NET Core 9 application built within a Microservice for managing orders, products and customers.

## Features

- Order processing and tracking

## Architecture

- Microservice (Clean Architecture)

## Tech Stack

- ASP.NET 9
- EF Core 9
- SQL Sever(InMemory)
- Docker
- Azure ContainerApp
- Azure Service Bus
- Github Workflow (CI/CD)
